## 搬瓦工套餐升级是什么

先说清楚这件事的逻辑。搬瓦工的套餐升级，不是重新买一台 VPS——是在原有服务基础上补差价，换一个配置更高或线路更好的套餐，原来的 IP、数据、系统设置全部保留，不需要重装，通常几分钟就能完成。

升级费用的计算公式是：**（新套餐价格 − 旧套餐价格）× 剩余天数 / 付款周期 × 优惠码折扣**。只补按比例折算的差价，不需要重新付全款。

## 后台升级按钮在哪里（新版界面路径）

很多人找不到升级入口，是因为按照旧教程里的截图去找，界面已经对不上了。新版后台的升级路径是这样的：

1. 登录搬瓦工后台（Client Area）
2. 顶部导航栏点 **Services → My Services**
3. 找到你要升级的 VPS，点右侧的**齿轮图标 ⚙️**
4. 在弹出菜单里选择 **Upgrade**
5. 进入升级页面后，选择目标套餐，点击 **Upgrade to this plan**，支付差价完成

注意：有些老版后台的路径是 Manage → Management Actions → Upgrade/Downgrade，跟新版位置不一样。如果你的后台还是旧版界面，按这个路径走。

## 搬瓦工后台没有升级选项的四种原因

### 原因一：当前套餐已经是该系列最高配置

这是最常见的情况。

搬瓦工的升级列表只显示**比你当前套餐更高配置**的方案。如果你已经买了该系列最高配置，升级列表就会是空的，看起来就像"没有升级选项"。

解法：考虑切换到另一个套餐系列。比如从 KVM 套餐切换到 CN2 GIA-E 套餐，就能在升级页面看到新的选项列表。

### 原因二：限量版套餐有升级限制

这个坑很隐蔽。

搬瓦工的套餐分**常规版**和**限量版**两类。限量版价格低、性价比高，但对升级路径有限制——通常只能升级到同系列的更高档限量版，不能跨系列升级到常规版。

举个例子：CN2 特价促销版只能升级到更高配置的 CN2 特价促销版，KVM 常规版只能升级到更高配置的 KVM 常规版。两者之间不能互相切换（部分新版本已开放例外，但不稳定）。

遇到这种情况，如果升级列表确实是空的，就需要提交工单请客服确认是否有可用的升级路径。

### 原因三：账户有未支付账单

搬瓦工后台有一个不太显眼的规则：如果账户存在未支付的账单，升级功能会被暂时锁定。

解法很简单：进入 Billing 页面，把欠款结清，升级按钮就会重新出现。

### 原因四：香港套餐和部分高端套餐无自助升级通道

香港 CN2 GIA 套餐（Ultra 系列）在大部分情况下没有自助升级选项，因为这个系列本身就是当前最高档次，没有更高的档位可以升。

如果你买了香港套餐并想调整配置，唯一的办法是提交工单联系官方客服。

## 提交工单请客服帮你升级（操作模板）

有些情况下自助升级确实走不通，提工单是正规路径，不复杂。

在搬瓦工后台：**Client Area → Support → Tickets → Open New Ticket**

选 **Billing** 类型，内容大致这样写：

> Subject: Help me upgrade my VPS plan
>
> Hi, I'd like to upgrade my current VPS to [目标套餐名称]. My current service is [你的 VPS 名称/ID]. Please let me know if this is possible and the cost. Thank you.

目标套餐名称去官网的套餐列表里找，黄色加粗的英文就是。

客服响应通常在 24 小时内，升级过程中 IP 和数据保留。

## 升级后数据和 IP 会变吗

不会丢数据。

这个问题被问得最多。搬瓦工的在线升级是在线完成的，系统只调整配置参数，不会格式化硬盘或者重装系统。实际体验下来，升级整个过程 VPS 会重启一次，大概 3-5 分钟，重启完新配置就生效了。

IP 地址是否变化要分情况：

- 在同一机房内升级 → IP 不变
- 跨机房升级或切换到完全不同的系列 → IP 可能变化

所以升级前可以在 KiwiVM 控制面板里做个快照备份，万一出意外也有保底。30 天退款政策也还在，新购用户有足够时间测试。

## 降级怎么办

顺带说一下。

目前搬瓦工**不支持自助降级**。后台没有降级的自助操作路径，官方页面上的按钮也只显示比当前套餐更高的方案。

想要降级，只能提交工单请客服人工处理，但成功率不高，部分促销套餐官方直接说不支持降级。如果当前套餐用不完、配置过剩，通常更划算的做法是等到期后重新购买一个低配套餐。

## FAQ

**Q：后台 My Services 页面显示了 VPS，但点开来没有 Upgrade 选项，是账号问题吗？**

A：不是账号问题。有两种可能：一是你用的新版后台，升级入口在右侧齿轮图标里，不是直接在 Manage 页面；二是你当前的套餐在该系列内已经是最高配置，系统没有更高的选项可以显示。建议先检查齿轮图标，再对照官网套餐列表看看有没有更高配置的同系列套餐存在。

**Q：升级套餐时能用优惠码吗？**

A：可以。购买时没用优惠码的，升级页面可以在 Promotional Code 框里补填。购买时用了旧优惠码的，也可以在升级时换成当前折扣力度更大的优惠码——点 "Don't use Promotional Code" 取消原来的，再输入新的。

**Q：限量版套餐（比如 THE PLAN、MINIBOX）可以升级吗？**

A：部分可以，部分不行。搬瓦工不定期会开放限量版套餐之间的升级通道，比如 MiniBox 可升级到 BiggerBox，BiggerBox 可升级到 PowerBox。但这取决于当时是否开放，不是一直有效。建议在后台实际查看升级列表，有的话直接操作，没有的话提工单确认。

**Q：搬瓦工后台登录不上去，Session Expired 怎么处理？**

A：清除浏览器里搬瓦工相关的 Cookie，重新登录。这是个已知的小毛病，不影响账户和 VPS 状态。

**Q：现在应该买哪个套餐？**

A：有明确需求的话：入门学习/轻量建站选 KVM 套餐 $49.99/年；需要速度和机房灵活性选 CN2 GIA-E 套餐，季付 $49.99；对延迟要求极高、预算够的才考虑香港套餐。不知道选哪个就先买 CN2 GIA-E，用了觉得不对还有 30 天退款托底。
